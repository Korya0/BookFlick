## 🚀 Key Features

<details>
<summary><b>📚 Books Explorer</b></summary>
<br>

- Browse popular and top-rated books.
- Interactive slider carousel for featured titles.
- Smooth animations and fade-in effects.

</details>

<details>
<summary><b>📖 Detailed Insights</b></summary>
<br>

- Comprehensive book details including synopsis and ratings.
- Author information and publication details.
- Elegant UI for readability.

</details>

<details>
<summary><b>💾 Offline Library</b></summary>
<br>

- Save books to your favorites locally (powered by Hive).
- Heart ❤️ animations for interactions.
- Access your saved books even without an internet connection.

</details>

<details>
<summary><b>🔍 Smart Search</b></summary>
<br>

- Real-time search results powered by Google Books API.
- Instant feedback as you type.
- Easy navigation to book details from search results.

</details>

<details>
<summary><b>🌙 Theme Support</b></summary>
<br>

- Seamless toggle between Light and Dark modes.
- Consistent design across both themes.

</details>

## 🛠️ Tech Stack

<hr style="height:1px;border:none;color:#333;background-color:#333;" />

- **Framework**: `Flutter`
- **Language**: `Dart`
- **Database**: `Hive (Local Storage)`
- **Cloud Interface**: `Google Books API`
- **State Management**: `Cubit (Bloc)`
- **Dependency Injection**: `GetIt`
- **Animations**: `Hero, Rive`

## 💻 Getting Started

<hr style="height:1px;border:none;color:#333;background-color:#333;" />

1. **Clone the repository**
   ```bash
   git clone https://github.com/Korya25/Bookna-app.git
   ```
2. **Install dependencies**
   ```bash
   flutter pub get
   ```
3. **Run the project**
   ```bash
   flutter run
   ```
